# The Path of Light

A polished first-person historical adventure and visual-novel prototype for a respectful Shia Islamic learning game. It combines street-level exploration, graphic-adventure investigation, branching dialogue, and source literacy without violence or depictions of sacred figures.

## Run

Open `index.html` directly, or serve the folder locally:

```powershell
cd C:\ProjectX\shia-path
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Prototype features

- First-person mouse and `WASD`/arrow-key camera movement
- `Q` Scholar’s Focus mode for discovering environmental clues
- Visual-novel dialogue with Ustadha Maryam, a fictional librarian
- Graphic-adventure evidence collection and branching knowledge choices
- Persistent Insight/Trust progress with `localStorage`
- Journal, source library, and learning principles
- Responsive layout, keyboard focus, and reduced-motion support

## Content safeguards

- Fictional city and architecture; no recreation of a holy site
- No visual depiction of Allah, prophets, Fatima al-Zahra, or the Imams
- Source notes separate educational summaries from legal rulings
- Production content should be reviewed by qualified Twelver Shia scholars and clearly identify school, edition, grading method, and scholarly disagreement

## Art asset

`assets/city-of-light.png` was generated specifically for this project with the built-in image generation tool. Prompt: an original top-down fictional Middle Eastern-inspired city at blue hour with a glowing route through a library, community district, garden, market, and observatory; midnight navy/lapis/teal/gold; no people, sacred figures, scripture, combat, weapons, real holy sites, logos, or text.
