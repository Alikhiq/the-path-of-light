# The Path of Light — refined concept

_Saved 2026-07-12. Ideation via idea-refine._

## Problem Statement
How might we let a 7-year-old explore a living Baghdad and *fall into* learning how
Shia scholars verify truth — then keep pulling them deeper as they grow?

## Recommended Direction
A mobile-first browser game (installable PWA), tap-to-explore ("node-hop") open
Baghdad, telling one story that scales from age-7 guided play to expert-level
source-investigation. Public and shareable by link, with an in-game suggestion box
feeding a real database so the game can grow from player feedback. Built in web tech
(Claude + agents) and deployed free.

Unreal Engine 5 is **deferred** to a possible 3D Phase 2 — only after the web game
proves fun and finds an audience. Starting in UE5 would kill the 1-month goal, can't
be co-built by the AI, and is the hardest possible path to mobile + public release.

## Key Assumptions to Validate
- [ ] A 7-year-old has fun tapping through node-hop scenes — test: one polished scene on a phone
- [ ] The easy→hard ramp keeps older kids engaged — test: chapter 1 easy vs chapter 3 hard
- [ ] Content survives scholarly review — test: editable content file + citations, reviewed first
- [ ] Parents will share a link — test: deploy chapter 1, watch reactions

## MVP Scope (1-month vertical slice)
**In:** mobile PWA shell · Baghdad hub map · 3 explorable chapters (easy / mid / hard) ·
easy-mode lantern guide + no-fail · dialogue + clue investigation · insight/trust
progress · suggestion box → database · deploy to a live URL

**Out:** 3D walking · combat · multiplayer · accounts/login · audio voiceover (stretch)

## Chapters (chosen: Verify → Chains → Chronology)
1. **The Unbroken Chain** (easy) — a saying sounding wise is not proof; trace its source and chain.
2. **Names in the Chain** (mid) — every narrator is a person you must know: honest? precise? did they meet?
3. **Broken by Time** (hard) — dates expose fake chains; if two narrators never overlapped, the chain breaks.

## Not Doing (and Why)
- Unreal / native 3D now — kills the 1-month goal, can't be co-built by AI, no audience yet
- App-store release — a PWA gives the app feel with none of the gatekeeping
- Hadith grading *by the game* — it teaches method only; real rulings need qualified scholars
- Depicting sacred figures (Allah, prophets, Fatima al-Zahra, the Imams) — stays out, permanently

## Content safeguards
- Fictional city, characters, and narration; no recreation of a real holy site
- Separates educational method from legal rulings; names uncertainty honestly
- Production content to be reviewed by qualified Twelver Shia scholars, identifying
  school, edition, grading method, and scholarly disagreement
