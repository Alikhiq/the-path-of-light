# Scholar Review Packet — The Path of Light

**Status: PENDING REVIEW. Do not treat any claim below as verified until a qualified
Twelver Shia scholar signs off in the table at the end.**

This game teaches the *method* of checking a report — it never grades a real hadith
and never issues a ruling. That guardrail only holds if every educational statement in
the game is accurate, precisely worded, and framed honestly. This document lists every
such statement, what a reviewer should check, and where corrections go.

All quoted text below is copied from [`content.js`](../content.js). Corrections must be
made to the quoted strings in that file only (structure/keys unchanged) — see
[`scholar-onboarding.md`](scholar-onboarding.md).

## How to review each item
For every claim, confirm all six:
1. **Concept** — is the verification principle correct?
2. **Terminology** — are the Arabic terms (and their transliteration) used correctly?
3. **Citation** — where a source is named, is the reference and edition right?
4. **Disagreement** — is a point of legitimate scholarly disagreement flattened into a false absolute?
5. **Sanctity** — no sacred figure (Allah, the Prophet, Fatima, the Imams) is depicted, nor quoted as issuing a real saying/ruling.
6. **Age framing** — the simplification for age ~7 does not become misleading.

Mark each **OK / Fix / Discuss** and note the correction.

## E. Open questions for the review panel
1. **Naming Imam al-Sadiq (C1.1):** keep as the example, or generalize to "an Imam"? No saying is attributed to him; he is only named.
2. **Absolutes for age 7 (C2.4, D3.5):** where the easy framing states a rule flatly ("an unknown link means the chain cannot be relied upon"; "impossible ⇒ fabricated"), is that acceptable simplification or does it need a hedge?
3. **No real narrators by design:** the game deliberately uses fictional, unnamed narrators and objects. Confirm this is the right choice for a children's product (vs. teaching with real, cited rijal cases at the hard tier).
4. **Grading vocabulary:** *sahih / hasan / muwaththaq* are used in the Twelver sense (C2.6/F4). Confirm no drift toward Sunni usage.
5. **Terminology transliteration:** agree one convention (e.g., *thiqa*, *majhul*, *ittisal*, *matn*, *sanad*, *dabt*, *wad'*) and apply consistently.
6. **Editions/citations:** if any real source is cited in future content, fix the edition and page before release.

## F. Correction workflow
- Edit only the quoted English text in [`content.js`](../content.js); keep every key, comma, and structure. See [`scholar-onboarding.md`](scholar-onboarding.md).
- Terminology or framing that recurs (foundational items) should be fixed everywhere it appears.
- After edits, a second reviewer re-checks the changed items.
